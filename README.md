# Album

The purpose was to create a website without using any frameworks, developing a boilerplate for future use.

## Getting Started

### Dependencies

* nodejs
* prismic (cms)

### Installing

```
npm install
```

### Executing program

* Start Express server
```
npm run be:build
```

* Start FE development server
```
npm run fe:development
```

* Build FE sources
```
npm run fe:build
```
